# Atomic-React

An example app for interacting with [`atomic-server`](https://github.com/joepio/atomic).

Goal: Edit an [Atomic Resource](https://docs.atomicdata.dev/core/concepts.html) from a browser!

## Usage

Run using `npx`:

```
npx snowpack dev
```

Or `yarn` / `npm`:

```sh
yarn
yarn dev
yarn build
```

## Linting

`yarn prettier --write .`

## Testing

`yarn test`
